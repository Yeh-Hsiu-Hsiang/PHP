### **這裡是 PHP 初學者練習**  

> **常用語法**  

* 基本輸出  
```echo``` 輸出一個或者多個字串  
```print``` 只能列印出簡單型別變數的值(如int,string)   

```  
echo "Hello, World"
print "Hello, World"
```

* 輸出陣列  
```print_r``` 可以列印出複雜型別變數的值(如陣列,物件)    

* 宣告變數  
```$變數名稱```  

* 設定函數  
```  function(輸入值){}```  

* 換行  
```echo "<br>";```  

* 清單  
```  
<ul>
  <li> </li>
  <li> </li>
</ul>
```  

* 分隔線  
```echo "<hr/>";```  

* 印出變數的相關資訊(ex:變數種類or值)  
```var_dump(要判斷的變數)```  
``` ex: var_dump($title)```
